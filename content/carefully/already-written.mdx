# already written

funny how I don't need to dig deep  
to find thoughts so relatable,  
thoughts that were once my own.

no need to write anymore,  
no need to repeat myself.

I've been a mess for quite a while, yet the liar inside of me,  
cunning as always,  
tames my wishes.
